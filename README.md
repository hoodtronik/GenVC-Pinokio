# GenVC Pinokio App

This is a [Pinokio](https://pinokio.computer/) installer and wrapper for [GenVC](https://github.com/caizexin/GenVC), a Self-supervised Generative LM-based Voice Conversion system.

## About GenVC
[GenVC](https://github.com/caizexin/GenVC) is an open-source, language model-based zero-shot voice conversion system that leverages self-supervised training and supports streaming voice conversion.
- Original Repository: [caizexin/GenVC](https://github.com/caizexin/GenVC)
- Paper: [Self-Supervised LM-Based Zero-Shot Voice Conversion](https://arxiv.org/abs/2502.04519)

## Installation
You can install this app directly in the [Pinokio browser](https://pinokio.computer/) using the URI for this repository.
1. Open Pinokio
2. Go to `Discover` and paste the URL of this repository.
3. Click `Download` to install the dependencies and models automatically.

## Usage
Once installed, click **Start** in Pinokio. A Gradio Web UI will launch, allowing you to run:
- Non-streaming inference
- Streaming inference
Using either the `GenVC_small` or `GenVC_large` models.
